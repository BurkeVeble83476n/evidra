# contract: v1.3.0

Evidra — Flight recorder for AI infrastructure agents.
It measures operational reliability across CI pipelines, scripts, and AI agents; it does not block operations.
Evidra speaks MCP: any MCP-capable AI agent can report executions and deliberate refusals to Evidra out of the box.


Default workflow for infrastructure tasks:
1) use `run_command` for investigation and most infrastructure work
2) use `describe_tool` if you need the full schema for deferred explicit-control tools
3) use `prescribe_smart` + `report` explicitly only when you need tighter control; use `prescribe_full` only when enabled and you have artifact bytes

Mutation examples (must use prescribe/report):
- kubectl apply/delete/patch/create/replace/rollout restart
- helm install/upgrade/uninstall/rollback
- terraform apply/destroy/import


Read-only examples (no prescribe/report):
- kubectl get/describe/logs/top/events
- helm list/status/template
- terraform plan/show/output


Critical invariants:
- Every infrastructure mutation must be recorded as evidence, either automatically via run_command or explicitly via prescribe/report.
- If you use prescribe_smart or prescribe_full, do not execute until the prescribe call returns ok=true with prescription_id.
- Every explicit prescribe must have exactly one report.
- Always include actor.skill_version (set to this contract version) on explicit protocol calls.


Rules:
- Default to `run_command` for investigation and most infrastructure mutations.
- Mutations executed through `run_command` are recorded automatically; verify each fix before making another change.
- Use `describe_tool` to inspect the full schema for `prescribe_smart` or `report` when you need explicit control.
- Use `prescribe_full` only when the server exposes it and you have artifact bytes.
- Report explicit-control failures with non-zero exit_code.
- If execution is intentionally declined in an explicit flow, report verdict=declined with a trigger and short operational reason.
- On retry in an explicit flow, call a prescribe tool again for each attempt.
- Do not use explicit prescribe/report for non-infrastructure tasks (docs, coding, analysis).
